# dotfiles manager

```
git clone https://github.com/Senpai-10/dotm.git
cd dotm
```

## **installing the cli version**

```
sudo make install-cli
```

#### usage

`$ dotm`

## **installing the gui version**

```
sudo make install-gui
```

#### usage

`$ dotm-gui`
